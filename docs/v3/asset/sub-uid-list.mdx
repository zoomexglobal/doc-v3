Query the sub UIDs under a main UID

:::caution
Can query by the master UID's api key **only**
:::

### HTTP Request
GET `/private/v1/asset/sub-member/member-ids`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|sub_user_id |array&lt;string&gt; |All sub UIDs under the main UID |

---

### Request Example

``` shell
GET /private/v1/asset/sub-member/member-ids?timestamp=1724491676233&api_key=0b2XYJpi8bi9tZPhft&recv_window=5000&sign=f70c48d470754baf537044038fa53543e71cd5292b5d68ff87be35fbaac8e669 HTTP/1.1
Host: openapi-testnet.zoomex.com
```

### Response Example

```json
{
    "ret_code": 0,
    "ret_msg": "OK",
    "result": {
        "sub_user_id": [
            "103741450",
            "103747845"
        ]
    }
}
```
